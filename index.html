Visual studio is an application for text based coding. We can use bootstrap also in this application. We have to download it into windows or mac.

Console screen is were we see codes that we dont see in the live screen. We print it by using variable and console.log() . We store what we want in the variable and then write it on console.log().Type the variable name in the brackets and Then right click, click inspect and go to console. We can see all the codes that we are unable to see in the live screen.
